# Developer Portfolio

Developer portfolio built with **Jekyll** and hosted on **GitHub Pages**. 

[Link to Live Site](https://your-jean-johnson-zwix.github.io)


## 🛠️ Local Development

To run this project locally, ensure you have Ruby, Jekyll, and Python 3 installed.

1.  **Clone the repo:**
```
git clone https://github.com/your-username/your-repo-name.git
```
2.  **Install dependencies:**
```
make deps
```
3.  **Start the server:**
```
make serve
```
4.  **View in browser:**
    Navigate to http://localhost:4000

## Makefile Commands

| Command | Description |
|---------|-------------|
| `make` | Build the Jekyll site |
| `make serve` | Serve locally with live reload |
| `make deps` | Install Ruby + Python dependencies |
| `make fetch` | Fetch all project READMEs from GitHub |
| `make clean` | Remove built files |
| `make help` | Show all available targets |
